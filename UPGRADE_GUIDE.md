# Руководство по обновлению QA Assistant

## Обзор изменений

В последнем обновлении были добавлены следующие функции:

1. Добавлена поддержка DeepSeek API
2. Создана отдельная страница с интеграцией DeepSeek API через Google Apps Script
3. Добавлена защита API-ключа через Google Apps Script
4. Обновлены документационные файлы

## Что нового

### DeepSeek версия
- Создана новая страница [deepseek.html](./deepseek.html) с аналогичным функционалом
- Использует Google Apps Script для безопасной интеграции с DeepSeek API
- Защищает ваш API-ключ от раскрытия на клиентской стороне

### Обновленные файлы
- [index.html](./index.html) - добавлена ссылка на DeepSeek версию
- [README.md](./README.md) - обновлена документация с информацией о новых возможностях
- [config.json](./config.json) - добавлена информация о поддержке нового провайдера
- Новый файл [DEEPSEEK_INTEGRATION.md](./DEEPSEEK_INTEGRATION.md) с инструкциями по настройке

## Как обновить существующую версию

Если вы уже развернули приложение и хотите обновить его:

1. Скачайте новые файлы:
   - [deepseek.html](./deepseek.html)
   - [deepseek-script.js](./deepseek-script.js)
   - [DEEPSEEK_INTEGRATION.md](./DEEPSEEK_INTEGRATION.md)
   - Обновленные версии [index.html](./index.html), [README.md](./README.md) и [config.json](./config.json)

2. Замените существующие файлы на новые

3. Если вы используете GitHub Pages, сделайте коммит с изменениями:
   ```
   git add .
   git commit -m "Добавлена поддержка DeepSeek API"
   git push origin main
   ```

## Настройка DeepSeek версии

Чтобы начать использовать DeepSeek API:

1. Следуйте инструкциям в [DEEPSEEK_INTEGRATION.md](./DEEPSEEK_INTEGRATION.md) для настройки Google Apps Script
2. Убедитесь, что у вас есть действительный API-ключ DeepSeek
3. Введите ID вашего Google Apps Script на странице [deepseek.html](./deepseek.html)

## Совместимость

- Существующая функциональность с OpenAI API полностью сохранена
- Все настройки и сохраненные API-ключи остаются в силе
- Новый функционал полностью изолирован от основной версии

## Возможные проблемы

Если у вас возникли проблемы после обновления:

1. Проверьте, все ли файлы были загружены на ваш хостинг
2. Убедитесь, что кэш браузера очищен (Ctrl+F5 для принудительной перезагрузки)
3. Проверьте, что файлы имеют правильные разрешения на чтение
4. Убедитесь, что при использовании DeepSeek версии, Google Apps Script развернут корректно

## Безопасность

Обратите внимание, что теперь у вас есть два варианта использования API:

1. **OpenAI (index.html)** - API-ключ хранится в локальном хранилище браузера
2. **DeepSeek (deepseek.html)** - API-ключ хранится на стороне Google Apps Script, что обеспечивает лучшую безопасность

Рекомендуется использовать DeepSeek версию, если безопасность API-ключа является приоритетом.
