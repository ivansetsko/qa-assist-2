# QA Assistant - Помощник для тестировщиков

QA Assistant - это веб-приложение, которое использует искусственный интеллект для помощи тестировщикам в создании тест-кейсов и баг-репортов.

## Функциональность

1. **Генерация тест-кейсов** - Опишите функционал, и ИИ создаст для вас подробные тест-кейсы
2. **Создание баг-репортов** - Опишите проблему, и ИИ поможет составить структурированный баг-репорт

## Требования

- Современный веб-браузер
- API-ключ от OpenAI (или другого ИИ-сервиса)

## Установка и запуск

1. Скачайте или склонируйте этот репозиторий
2. Откройте файл `index.html` в вашем веб-браузере
3. Введите ваш API-ключ и нажмите "Сохранить"

## Использование

### Получение API-ключа

Для использования приложения вам понадобится API-ключ от ИИ-сервиса. В настоящее время приложение настроено для использования OpenAI GPT, но может быть адаптировано под другие ИИ-сервисы.

1. Зарегистрируйтесь на [OpenAI](https://platform.openai.com/signup)
2. Перейдите в раздел API-ключей (https://platform.openai.com/api-keys)
3. Создайте новый API-ключ и скопируйте его

### Генерация тест-кейсов

1. Введите ваш API-ключ и нажмите "Сохранить"
2. В поле "Генерация тест-кейсов" опишите функционал, для которого вы хотите создать тест-кейсы
3. Нажмите кнопку "Сгенерировать тест-кейсы"
4. Результаты появятся в поле под кнопкой

### Создание баг-репортов

1. Введите ваш API-ключ и нажмите "Сохранить"
2. В поле "Создание баг-репортов" опишите проблему, которую вы наблюдаете
3. Нажмите кнопку "Создать баг-репорт"
4. Результаты появятся в поле под кнопкой

## Настройка под другие ИИ-сервисы

Если вы хотите использовать другой ИИ-сервис, измените соответствующие части в файле `script.js`, особенно функцию `sendToAI`.

## Безопасность

Ваш API-ключ сохраняется в локальном хранилище браузера. Никогда не делитесь этим ключом публично и не коммитьте его в репозиторий.
