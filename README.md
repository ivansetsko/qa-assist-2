# QA Assistant - Помощник для тестировщиков

QA Assistant - это веб-приложение, которое использует искусственный интеллект для помощи тестировщикам в создании тест-кейсов и баг-репортов.

## Функциональность

1. **Генерация тест-кейсов** - Опишите функционал, и ИИ создаст для вас подробные тест-кейсы
2. **Создание баг-репортов** - Опишите проблему, и ИИ поможет составить структурированный баг-репорт

## Доступные версии

Проект включает две версии с разными ИИ-провайдерами:

1. **OpenAI версия** (на [index.html](./index.html)) - использует OpenAI GPT-3.5-turbo
2. **DeepSeek версия** (на [deepseek.html](./deepseek.html)) - использует DeepSeek API через Google Apps Script

## Требования

- Современный веб-браузер
- API-ключ от OpenAI (для основной версии) или DeepSeek (для второй версии)

## Установка и запуск

1. Скачайте или склонируйте этот репозиторий
2. Откройте файл [index.html](./index.html) или [deepseek.html](./deepseek.html) в вашем веб-браузере
3. Введите ваш API-ключ и нажмите "Сохранить"

## Использование

### Основная версия (OpenAI)

1. Зайдите на [index.html](./index.html)
2. Введите ваш API-ключ OpenAI и нажмите "Сохранить"
3. Используйте поля для генерации тест-кейсов и баг-репортов

### DeepSeek версия

1. Зайдите на [deepseek.html](./deepseek.html)
2. Следуйте инструкциям в [DEEPSEEK_INTEGRATION.md](./DEEPSEEK_INTEGRATION.md) для настройки Google Apps Script
3. Введите ID вашего скрипта GAS и нажмите "Сохранить"
4. Используйте поля для генерации тест-кейсов и баг-репортов

### Получение API-ключа OpenAI

1. Зарегистрируйтесь на [OpenAI](https://platform.openai.com/signup)
2. Перейдите в раздел API-ключей (https://platform.openai.com/api-keys)
3. Создайте новый API-ключ и скопируйте его

### Получение API-ключа DeepSeek

1. Зарегистрируйтесь на [DeepSeek](https://www.deepseek.com/)
2. Перейдите в раздел API
3. Создайте новый API-ключ и скопируйте его

## Безопасность

Ваши API-ключи сохраняются в локальном хранилище браузера. Никогда не делитесь этими ключами публично и не коммитьте их в репозиторий.

Для версии с DeepSeek API-ключ хранится в Google Apps Script, что обеспечивает дополнительный уровень безопасности.
